创建一个属于自己的github账户， 1.申请账号 2.安装github 3.捆绑： (1).在命令行上写： ssh-keygen -t rsc -C "注册的邮箱" (2).根据控制台上的地址找到id_rsa.pub,将这个密钥放到GitHub上：在你的github账户上 -> settings -> SSH and GPG keys -> new ssh key (3).回到命令行那里：设置作者的基本信息（名字(user.name)、邮箱(user.email)）: git config -- user.name "你的GitHub用户名" 回车 git config -- user.email "你的GitHub邮箱" 回车 (4).看一下捆绑是不是成功了：(如果成功会有Hi-xxx) ssh -T git@github.com 4.创建一个新的项目 (1).在github网站上：点击右上角的加号 -> New repository -> 填写项目名称、项目描述，勾选README 5.克隆项目到本地 (1).打开命令行（控制台），进入到要存放项目的目录，右击点击 (2).在githu官网上，复制克隆的地址，回到控制台： git clone 粘贴那个克隆地址 回车 6.删除项目： 在GitHub官网上： 找到并进入项目 -> settings -> 滑到最后 Delete this repository(按钮) -> 输入删除的项目名称 -> 确定 6.上传本地项目到github官网 命令行（操作台）上： git add . 回车 git commit -m "你的注释" 回车 git push origin mastar
