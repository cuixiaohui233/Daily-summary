分支： 作用：多人协作分工 在公司里会这样共同开发一个项目 查看分支： git branch 新建分支： git branch 分支名 切换分支： git checkout 分支名 快捷方式：（快速新建并切换） git checked -b 分支名 注意： 新建分支的时候一定要看清楚，当前分支在哪里，如果不是复制模板（想要建分支的地方），要切换到复制模板上。 删除分支（已合并）： git branch -d 分支名 删除分支（未合并）： git branch -D 分支名 合并分支: git merge 要合并的分支名 查看未合并的分支： git branch --no-merged 查看已合并的分支： git brench --merged 注意： 在合并的时候，有可能会发生冲突，那么需要解决冲突： 1.将主干和分支都提交到版本库； 2.合并（git merge 分支名）； 3.git status (查看冲突)； 4.手动解决； 5.如果需要，可以把合并之后的分支删除
